Dashboard Guide
Learn how to navigate and use every feature
What is this dashboard?
This is the Mixta Africa Projects Tracker — a live, real-time dashboard for monitoring Lakowe Crossings and Lakowe Annexe. It syncs directly from your project spreadsheets every 5 minutes.
Data is live-synced
The dashboard automatically pulls the latest data from Google Sheets. You'll see a "Last synced" time at the top of the page. No manual refresh needed.
You're signed in to your department
Your Google account determines which project and department you can see. You only have access to your own department's data — other tabs are locked for privacy.
Works on mobile too
The dashboard adapts to your screen size. On mobile, use the hamburger menu (≡) at the top to switch between pages like Analytics, Milestones, and Trends.
Lakowe Crossings (Green)
The Lakowe Crossings project is displayed with a green theme. Click "Crossings" in the top navigation bar to switch to this project view. KPI tiles, charts, and tasks will update to show Crossings data.
Lakowe Annexe (Red)
The Lakowe Annexe project is displayed with a red theme. Click "Annexe" in the navigation to switch. Each project maintains its own task history, financial data, and session logs.
Session picker
Use the date/session dropdown at the top to browse historical meeting data. Each session represents one project meeting. The default view shows the most recent session.
Priority tiles — Account Balance & Offer Letters
The two gold-bordered tiles at the top right are the highest-priority KPIs. They change colour automatically: Red = critical, Amber = watch closely, Green = healthy.
Key Feature
Reading a task card
Each task card shows the Context/Result bar at the top (so you know WHY the task matters), then the action item below. The left-border colour shows the status: Green = On Track, Red = Delayed, Gold = On Hold, Blue = Done.
Updating task status
Click any task card to expand it, then use the status buttons to mark it as Done, Delayed, or On Hold. Your update is saved immediately and synced to the shared database.
Critical Alert Zone
The red strip below the KPI tiles collects all Delayed tasks from every department in one place. This is the CEO's first-look summary — check it before every meeting.
Trend badges
Some tasks show a "Repeated" badge — this means the same action item has appeared in multiple sessions without being marked as Done. These are escalation signals worth reviewing.
Financial Progress Trend chart
This chart tracks two lines over time: Sales Value (dashed blue) and Total Inflow (solid, project colour). The gap between the two lines represents outstanding receivables — the closer they are, the healthier the cash position.
Spreadsheet Targets Status
The card next to the chart shows this month's targets from the spreadsheet. Green = Done, Orange = Ongoing, Red = At Risk. Scroll within the card to see all targets.
Account Balance Trend (CVI)
Below the charts is the Commercial Velocity Indicator — it shows the trend in account balance, inflow rate, and whether the project is in a "cash drought". Red drought strips signal periods of no inflow.
Hovering for details
Hover over any point on a chart to see an exact value tooltip. The crosshair will snap to the nearest data point for both lines simultaneously.
Send Task Reminders
Use the envelope icon in the toolbar to compose and send email reminders to any department. Select the project, department, and Test or Live mode — Test sends only to the admin address for preview.
Monthly Report
The report icon generates a structured Word document (.docx) monthly summary for executives. Select the project, click Generate, and download the report for distribution.
AI Project Assistant
The chat bubble at the bottom opens an AI assistant trained on your project data. Ask it questions like "What are the most delayed tasks?" or "Summarise Annexe progress this month."
Team Updates
The message icon opens the Team Updates feed — a WhatsApp-style chat log where team members post progress notes, photos, and file attachments by department.
Admin access
Admin features are only visible to the admin account. If you're an admin, use the lock icon in the toolbar and enter the admin password to unlock the Admin Panel.
Health monitoring
The Admin Panel has a Health tab that runs 9 system checks — Firebase, Sheets sync, GAS relay, Groq API, and more. A warning banner appears automatically if any critical system fails in the last 30 minutes.
Sheets diagnostics
Admins can inspect raw data from Google Sheets in the Admin Panel to debug sync issues. The Error Log tab shows a rolling history of any sync failures with timestamps.
